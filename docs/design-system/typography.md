# EcoLens AI Typography

## Font Family
- Primary font: Inter
- Use Inter for all product UI, dashboard content, and product marketing surfaces

## Type Scale
### Display
- Size: 48px
- Line height: 56px
- Weight: 700
- Letter spacing: -0.02em

### H1
- Size: 32px
- Line height: 40px
- Weight: 700

### H2
- Size: 24px
- Line height: 32px
- Weight: 600

### H3
- Size: 20px
- Line height: 28px
- Weight: 600

### Subtitle
- Size: 16px
- Line height: 24px
- Weight: 500

### Body
- Size: 14px
- Line height: 22px
- Weight: 400

### Caption
- Size: 12px
- Line height: 18px
- Weight: 400

### Button
- Size: 14px
- Line height: 20px
- Weight: 600

## Usage Guidelines
- Display is reserved for hero moments and major dashboard titles
- H1 and H2 are used for page headers and section titles
- H3 is used for card headings and grouped content titles
- Subtitle supports descriptive copy and summary text
- Body is the default for paragraphs and labels
- Caption is used for metadata, helper text, and secondary details
- Buttons should use the Button size and weight consistently

## Text Treatment
- Prefer concise, direct copy
- Maintain strong spacing between paragraphs and sections
- Use semantic emphasis sparingly for premium clarity
