# EcoLens AI Component Guidelines

## 1. Buttons
### Variants
- Primary
- Secondary
- Outline
- Ghost
- Danger
- Success
- Loading Button

### Button Behavior
- Primary is used for the main action in a flow
- Secondary supports supporting actions
- Outline is suitable for less prominent but still important actions
- Ghost is used for low-emphasis actions
- Danger and Success are reserved for destructive or affirmative outcomes
- Loading buttons should clearly show progress state

## 2. Cards
### Supported Card Types
- Analytics Card
- Metric Card
- Glass Card
- Information Card
- Chart Card
- Recommendation Card

### Card Principles
- Use elevated surfaces with rounded corners
- Maintain consistent padding and spacing
- Support compact and rich states depending on context
- Use subtle borders and soft shadows for depth

## 3. Navigation
### Navbar
- Minimal, premium, and unobtrusive
- Support search, notifications, and profile access

### Sidebar
- Persistent for dashboard and workspace navigation
- Group navigation by sections and priorities

### Breadcrumb
- Clear path structure for nested views

### Profile Menu and Notification Panel
- Use compact, layered, and polished dropdown patterns

## 4. Inputs
### Supported Inputs
- Search
- Upload
- Dropdown
- Date Picker
- Checkbox
- Radio
- Switch
- Slider

### Input Principles
- Keep controls clear, rounded, and consistent
- Use labels and helper text with strong visual hierarchy
- Support validation states with subtle and meaningful feedback

## 5. Tables
### Table Standards
- Modern and lightweight, not overly dense
- Support sorting, filtering, and pagination
- Use row hover and selection states for clarity
- Keep headers visually distinct and readable

## 6. Charts
### Supported Chart Types
- Line Chart
- Bar Chart
- Pie Chart
- Area Chart
- Heatmap

### Chart Principles
- Use a restrained palette and readable labels
- Highlight the most important metric clearly
- Avoid clutter and over-visualization

## 7. Dashboard Composition
### Recommended Sections
- Top KPI Cards
- Charts Section
- Recent Activity
- AI Insights Panel
- Recommendation Panel
- Footer

### Layout Guidance
- Keep the dashboard structured and scannable
- Use consistent card spacing and a strong grid rhythm
- Maintain balance between analytics density and visual calm
