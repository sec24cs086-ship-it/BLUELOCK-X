# EcoLens AI

## Project Architecture Blueprint
This repository contains the initial architecture plan for EcoLens AI, an AI-powered carbon footprint receipt platform.

## Scope
- Frontend architecture for React 19, TypeScript, Vite, Tailwind CSS, shadcn/ui, Framer Motion, React Router, and Recharts
- Backend architecture for FastAPI, Python, SQLAlchemy, PostgreSQL, and JWT authentication
- AI processing architecture for OCR and carbon estimation workflows

## Repository Structure
- docs/ARCHITECTURE.md: high-level architecture and folder structure
- docs/ROADMAP.md: phased development plan
- frontend/README.md: planned frontend architecture
- backend/README.md: planned backend architecture
- .github/ISSUE_TEMPLATE/: issue templates for future work
